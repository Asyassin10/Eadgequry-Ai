PROBLEM:
- UnsupportedClassVersionError: class file version 65.0 (Java 21) vs 61.0 (Java 17)
- Services compiled with Java 21 but containers had Java 17

SOLUTION:
=========
All Containerfiles now use: FROM eclipse-temurin:21-jdk-alpine

STANDARDIZED FORMAT:
--------------------
FROM eclipse-temurin:21-jdk-alpine
WORKDIR /app
COPY target/[service]-0.0.1-SNAPSHOT.jar app.jar
EXPOSE [port]
ENTRYPOINT ["java","-jar","/app/app.jar"]
